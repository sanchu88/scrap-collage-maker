# Scrap Collage Maker

Android・Windowsのブラウザで使える、スクラップブック風コラージュ作成Webアプリです。

## 主な機能

- 複数写真の読み込み
- サムネイルのドラッグ並べ替え
- 写真ごとの切り取り位置・拡大率調整
- 通常／大枠1／大枠2レイアウト
- 文字カードの追加
- 背景色、白フチ、影、マスキングテープ
- スタンプ配置
- PNG／JPEG保存
- PWA対応

画像処理はブラウザ内で行われ、選択した写真をサーバーへ送信しません。

## GitHub Pages

Settings → Pages → Build and deployment で、Sourceを `Deploy from a branch`、Branchを `main`、Folderを `/(root)` に設定してください。
